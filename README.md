# PetGen

Sistema de gerenciamento de pets desenvolvido para facilitar o cadastro, gerenciamento e acompanhamento de informações relacionadas a animais e seus responsáveis.

---

## 🚧 Status do Projeto

O projeto ainda está em fase inicial de desenvolvimento.  
Novas funcionalidades, documentação e melhorias serão adicionadas gradualmente.

---

## 🛠 Tecnologias Utilizadas

### Front-end

- Next.js
- TypeScript
- Tailwind CSS

### Back-end

- Laravel
- PHP

### Banco de Dados

- MySQL

### Infraestrutura

- Docker
- Docker Compose

---

## 📁 Estrutura do Projeto

```bash
PetGen/
├── backend/   # API Laravel
├── frontend/  # Aplicação Next.js
└── docker/    # Configurações de containers
```

---

## ⚙️ Requisitos

Antes de iniciar o projeto, certifique-se de possuir instalado:

- Docker
- Docker Compose
- Git

---

## 🚀 Como Executar o Projeto

### 1. Clone o repositório

```bash
git clone <URL_DO_REPOSITORIO>
```

### 2. Acesse a pasta do projeto

```bash
cd PetGen
```

### 3. Suba os containers

```bash
docker compose up -d
```

---

## 📌 Funcionalidades Planejadas

- Cadastro de pets
- Cadastro de usuários
- Histórico de atendimentos
- Controle de vacinação
- Upload de imagens
- Dashboard administrativo
- Sistema de autenticação

---

## 📖 Objetivo

O objetivo do sistema é centralizar o gerenciamento de informações de pets em uma aplicação moderna, escalável e de fácil manutenção.
